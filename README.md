### API Documentation


#### Project Routes

| Method | Endpoint                        | Description                                        |
|--------|---------------------------------|----------------------------------------------------|
| GET    | `/projects`         | Returns all projects       |
| GET    | `/projects/:id`         | Returns the information for a project.       |
| PUT    | `/projects/:id`         | Modify an existing project.                   |
| DELETE | `/projects/:id`         | Delete a project.                            |
